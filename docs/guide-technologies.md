# Content Editor Guide: Technology Library

The Technology Library is a reusable database of tools, frameworks, and languages you use in your projects.

## Technology Fields

| Field | Type | Description |
| :--- | :--- | :--- |
| **Title** | String | Name of the tech (e.g. React, Node.js). |
| **Slug** | Slug | Unique identifier for sorting and queries. |
| **Tech Category** | List | Select category: Frontend, Backend, Mobile, Cloud, AI, etc. |
| **Logo** | Image | Visual icon. **Vector SVG** is highly recommended. |
| **Website URL** | URL | Official website link for the technology. |
| **Sort Order** | Number | Rank for tech stack displays. |
| **Active** | Boolean | Toggle visibility across all tagged projects. |

## Editing Tips
- **Consistency**: Use the same category names for similar techs (e.g. React and Vue should both be `Frontend`).
- **Logos**: Use high-quality, transparent logos to maintain a premium look in the portfolio "Tech Stack" section.
